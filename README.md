# OTP demo

For learning how OTP flow should be

## Tech stack

- [Bun](https://bun.sh/)
- [Elysia](https://elysiajs.com/)
- [Twillio](https://www.twilio.com/en-us)
- [Resend](https://resend.com/home)
- [Cloudflare](https://www.cloudflare.com/)

## Development

1. Install package

```
bun i
```

2. Start server

```
bun run dev
```

3. Open localhost

```
http://localhost:<YOUR_PORT>/
```
